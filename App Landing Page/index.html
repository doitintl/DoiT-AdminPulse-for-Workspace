This comprehensive checklist, housed in Google Sheets™ and maintained by Dustin Christlieb, is tailored for Super Admins seeking a security review of Google Workspace and Cloud Identity environments.

Drawing inspiration from the renowned GAM, this application harnesses the power of the Admin SDK to collect asset data seamlessly and integrate it into a Google Sheet. Currently, it focuses on read-only functionality, ensuring no modifications to API settings.

Delve into the appscript code on our GitHub repository for insights into its architecture and potential future developments as a Google Sheets™ Editor add-on.

Access a view copy of the live checklist and appscript code for a comprehensive overview.

For personalized assistance or to review your findings, feel free to reach out to Dustin Christlieb at workspace-security@doit.com.

Key Features

Comprehensive Security Settings

Access a detailed checklist of settings with direct links to Google's Admin Console for easy configuration.

User Audit

Track user activities including last sign-in date, status, 2SV enrollment, and admin privileges.

Domain & DNS Audit

Ensure your domain conforms to best practices by auditing MX, SPF, DKIM, and DMARC records.

Google Group Settings Audit

Review Google Group™ settings to ensure proper access controls.

Google Groups Membership Audit

Review Google Group™ memberships for both internal and external accounts.

Mobile Device Audit

Monitor mobile devices connected to your Google Workspace for security compliance.

License Audit

Track licenses to optimize usage and reduce costs.

Shared Drive Settings Audit

Review settings for Shared Drives to ensure data security.

OAuth Token Audit

Review OAuth tokens for third-party app access to your Google Workspace.

App Passwords Audit

Review user configured app passwords for less secure apps.

Organizational Unit Audit

Review organizational units for effective organization management.
